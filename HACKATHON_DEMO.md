# 🎤 Hackathon Demo Script

## 2-3 Minute Live Demo Plan

### Setup (Before Demo)
1. ✅ Have Freighter wallet installed with testnet account
2. ✅ Have a GitHub repo ready to submit
3. ✅ Have a second wallet with credentials (for verifier demo)
4. ✅ App running at `localhost:3000` or deployed URL

---

## 🎬 Demo Flow

### **Opening (15 seconds)**
> "EduProof Zero solves a massive problem: **resumes are lies, certificates show attendance not ability**. We've built a Web3 system that issues tamper-proof skill credentials backed by real evidence, verified by AI, and stored permanently on Stellar blockchain."

### **Part 1: Student Flow (60 seconds)**

**Step 1: Connect Wallet**
- Open `/student` page
- Click "Connect Freighter"
- Show wallet address connected
> "First, students connect their Stellar wallet. This becomes their identity."

**Step 2: Submit Evidence**
- Select "Python" skill
- Paste GitHub repo URL
- Click "Run AI Evaluation"
> "They submit real evidence - a GitHub repo, Google Doc, Loom video, or portfolio. Our AI evaluates it against a transparent rubric."

**Step 3: Show Results**
- Wait for evaluation (show loading state)
- Show score and feedback
> "The AI provides detailed feedback and a score. Notice the rubric is public - no black box here."

**Step 4: Mint Credential**
- Click "Mint Credential NFT"
- Show Freighter popup
- Approve transaction
- Show success message with transaction hash
> "If they pass, they mint a soulbound NFT credential. This is permanent, on-chain, and can't be faked."

### **Part 2: Verifier Flow (45 seconds)**

**Switch to Verifier Tab**
- Go to `/verifier` page
> "Now let's see the employer side. No login, no trust required."

**Search Wallet**
- Paste the wallet address from Part 1
- Click "Verify"
> "Any employer can paste a wallet address and instantly see all verified skills."

**Show Results**
- Display credentials with scores
- Click "View on Explorer" link
> "They see the skill, level, score, and can verify the transaction on Stellar Explorer. Everything is transparent and on-chain."

### **Closing (30 seconds)**

**Key Points:**
> "This solves real problems:
> - ✅ Skills backed by evidence, not certificates
> - ✅ Impossible to fake - it's on blockchain
> - ✅ Zero-trust verification - no centralized database
> - ✅ Works for self-taught developers, bootcamp grads, career switchers
> 
> The future of credential verification is here, and it's on Stellar."

## 🚀 Post-Demo Q&A Prep

**Q: How do you prevent cheating?**
A: Evidence is public, AI uses transparent rubrics, and credentials are on-chain. You can't fake a GitHub repo with real commits.

**Q: What about privacy?**
A: Only the wallet address is public. Evidence links are stored but can be private repos/docs if user chooses.

**Q: Who pays?**
A: Currently free on testnet. Future: Recruiters pay for API access, bootcamps pay to issue credentials.

**Q: Why Stellar?**
A: Fast, cheap, scalable. Perfect for credentials that need to be verified globally.
