## How to Take Screenshots

1. Run `npm run dev` to start the development server
2. Navigate to http://localhost:3000
3. Use your browser's screenshot tool or a tool like:
   - Windows: Snipping Tool or Win + Shift + S
   - macOS: Cmd + Shift + 4
   - Browser: DevTools screenshot or extensions
4. Save images with the exact filenames listed above
5. Place them in this folder

## Image Specifications

- **Format:** PNG (preferred) or JPG
- **Width:** 1920px or larger for best quality
- **Aspect Ratio:** 16:9 recommended
- **File Size:** Keep under 2MB per image for faster loading
